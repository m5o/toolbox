# lookup for l10n / i18n

Debug i18n lookup for l10n / i18n strings

## Installation

* copy `i18n_lookup.rb` to `app/config/initializers/i18n_lookup.rb`

## Usage

currently the variable `ENV['I18N_DEBUG']` is disabled. Modify if needed.
